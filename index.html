💱 Conversor de Monedas

Este es un proyecto hecho en Java que permite convertir entre diferentes monedas usando una API externa. Fue desarrollado como parte de mi proceso de aprendizaje en programación.

🔧 Tecnologías usadas

  • API de conversión de divisas
  • IDE: IntelliJ IDEA
